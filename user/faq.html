Administrasi Desa

Menambahkan Data Administrasi

1. Klik tombol +Tambah pada bagian sebelah kanan atas layar Anda.

2. Maka akan muncul form seperti di bawah ini. Anda wajib mengisi kolom yang mempunyai tanda bintang merah (*).


3. Jika telah selesai mengisi form tersebut, silakan klik tombol Simpan.

Mencetak Data Administrasi

1. Untuk mencetak Buku Administrasi Desa secara keseluruhan, langsung saja klik tombol Print pada sebelah kanan atas layar untuk masuk ke mode print.

2. Setelah Anda mengklik tombol Print , Anda akan memasuki mode print seperti gambar di bawah ini. .


Keterangan:
Kotak berwarna biru untuk print.
Kotak berwarna hijau untuk export file ke Excel.
Kotak berwarna merah untuk keluar dari mode print.

3. Untuk mencetak aturan tertentu pada buku peraturan desa anda bisa menggunakan filter tahun untuk memilah sesuai dengan tahunnya atau dengan menggunakan menu pencarian untuk mencari aturan yang lebih spesifik, seperti mencari berdasarkan nama peraturannya. Setelah data yang ingin anda cetak tampil di layar, barulah klik tombol print pada bagian kanan atas.

Manajemen Tanda Tangan

1. Pada menu utama klik ikon profil kemudian pilih Pengaturan.

2. Pilih menu manajemen tanda tangan.


3. Arahkan petunjuk Bisa Tanda Tangan ke kanan.

Pelayanan Desa

Menambahakan Data Penduduk

1. Klik pada menu Pelayanan Administrasi Kependudukan.

2. Setelah itu klik menu Biodata Penduduk.


3. Kemudian klik fitur layanan Formulir Kartu Keluarga.

4. Klik tombol +Tambah pada bagian sebelah kanan atas layar.

5. Maka akan muncul form seperti di bawah ini. Anda wajib mengisi kolom yang mempunyai tanda bintang merah (*). Jika telah selesai mengisi form tersebut, silakan klik tombol Simpan.

Mencetak Berkas Kependudukan

1. Untuk mencetak berkas kependudukan seperti Formulir Kartu Keluarga, langsung saja klik tombol Cetak di ujung kanan tiap daftar formulir kartu keluarga.

2. Setelah Anda mengklik menu print, Anda akan memasuki mode print seperti gambar di bawah ini.


Keterangan:

  • Kotak berwarna biru untuk mencetak.
  • Kotak berwarna hijau untuk mengarsipkan file.
  • Kotak berwarna orange untuk masuk ke mode pengaturan (tanda tangan, tanggal, jenis dan ukuran huruf, dan ukuran kertas)
  • Kotak berwarna merah untuk keluar dari mode print.

Website Profil

Menambahkan Isi Website

1. Pada menu utama pilih Web Profil Desa

2. Pilih menu sesuai dengan apa yang ingin ditambahkan seperti Berita Desa

3. Klik tombol +Buat baru untuk membuat berita baru pada web

4. Isi data berita yang akan dimasukkan

5. Klik tombol Simpan dan terbitkan atau Simpan Ke Draft

2. Pilih menu Pemerintahan

3. Isi data pada Form Pemerintahan

4. Klik tombol Simpan

Menambahkan Staf pemerintahan

1. Pada menu utama pilih Web Profil Desa

2. Pilih menu Staf Pemerintah

3. Klik tombol +Buat Baru

4. Isi data pada Form Pemerintahan

5. Klik tombol Simpan

FAQ

Apa Itu Digital Desa?

Digital Desa (Digides) adalah Sistem Informasi yang dirancang untuk menangani berbagai hal dalam pengurusan dalam desa. Mulai dari administrasi dan pelayanan desa menggunakan website ataupun aplikasi pada Android. Sehingga dapat membantu masalah pelayanan desa secara komputerisasi.

Mengapa Harus Bergabung Dengan Digides?

Di era modern saat ini kita tidak bisa lepas dari digitalisasi sehingga semuanya dilakukan melalui internet. Termasuk dengan pengurusan surat-surat ataupun administrasi sehingga diperlukan penunjang bagi masyarakat untuk mempermudah hal itu. Digides hadir dengan berbagai fitur yang dapat membantu masyarakat desa.

Keuntungan Menggunakan Digides?

Dengan bergabung bersama Digital Desa maka segala urusan administrasi, pelayanan masyarakat desa, pengurusan berkas perangkat desa, administrasi keuangan hingga administrasi pembangunan dan kas desa dapat diakses dengan mudah oleh perangkat desa maupun warga desa.

Fitur Apa Saja yang Dimiliki Digides?

Administrasi Umum mempermudah dalam pencatatan data dan informasi penduduk, Administrasi Keuangan untuk pencatatan dan informasi pelaksaanan pembangunan dan pemberdayaan masyarakat, Administrasi Kelembagaan untuk mencatat informasi mengenai penyelenggaraan kegiatan kelembagaan dalam desa.
